# Python Calculator

A simple command-line calculator built using Python that performs basic arithmetic operations using the `match-case` statement.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Menu-driven interface
* Input validation for menu selection
* Cross-platform screen clearing (Windows/Linux)

## Requirements

* Python 3.10 or later (required for `match-case`)


## Usage

```text
---------------- MENU ----------------
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Exit
```

Choose an option and enter two numbers when prompted.

### Example

```text
Enter your choice: 1
Enter a number: 10
Enter another number: 20

The Addition of 10 + 20 is 30
```
