# 🎮 Rock Paper Scissors

A simple command-line Rock Paper Scissors game built with Python. Play against the computer, compete over multiple rounds, and see who wins based on the final score.

## Features

* Play against the computer
* Random computer choices
* Multiple rounds support
* Score tracking
* Input validation and error handling
* Final winner announcement

## Requirements

* Python 3.x

## How to Run

```bash
python main.py
```

## Game Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
* If both players choose the same option, the round is a tie
